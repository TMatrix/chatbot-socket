# chatbot-socket

#### Start project:
1. Install dependencies:
```npm install```
2. Start server:
```npm start```

#### Enter chat:
Open browser tab on:
```localhost:5000```

#### Invoke chatbot:
Add to your message "@bot" and command:
```Save note title: "Your note title" body: "Note body"```
```Show note "Your note title"```
```Save note list```
```Delete note "Your note title"```
